#	领域语言

领域语言（domain-specific language）是应用于特定领域的语言。与通用程序语言相比，DSL 可能因其具体而更加复杂，也可能因其专门而更简单。通常我们所能接触到的领域语言，主要有：

*	软件配置语言，比如 VCL（Varnish Configuration Language）；
*	服务交互语言；
*	命令交互语言，比如 AWK；
