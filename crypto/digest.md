#	数据摘要

>	以管窥豹，时见一斑。

所谓数据摘要，并非简单地从数据全文中提取片断，而是经过特定算法处理，输出一段通常具有以下特征的数据短文：

*	固定长度
*	不可阅读
*	唯一

简单说，它就是一段“数据指纹”。常见的数据加密算法分为以下几类：

*	__Hash__  
	俗称“哈希”算法。

*	__MAC__, Message Authentication Code  
	消息认证码是由消息主文加上特定密钥作为输入，生成的数据摘要。

*	__HMAC__, Hash-based Message Authentication Code  
	HMAC 归属于 MAC，强调使用 Hash 算法对输入进行处理。
