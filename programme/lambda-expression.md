#	&lambda; 表达式

抛开复杂的数学定义，从感性的程序写作的角度看，Lambda 表达式的实质，是用极简的方式定义一个局部的、匿名的、通常是一次性使用的函数。

Python 较早支持 Lambda 表达式的主流编程语言：

```python
a = [ 1, 2, 3 ]
map(lambda x: x*x, a)
# OUTPUT: [ 1, 4, 9 ]
```
